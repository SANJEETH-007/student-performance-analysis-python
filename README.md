# 📊 Student Performance Analysis using Python

## 📌 Project Overview

This project analyzes the **Student Performance Dataset** using Python to explore the factors influencing students' final grades. It demonstrates the complete data analysis workflow, including data preprocessing, exploratory data analysis (EDA), statistical analysis, and data visualization.

This project was completed as **Task 1** of my **Data Science & Analysis with Python Internship**.

---

## 🎯 Objectives

* Load and explore the student performance dataset.
* Clean and preprocess the data.
* Analyze students' academic performance.
* Examine the relationship between study time and final grades.
* Compare average performance based on gender.
* Generate meaningful insights through data analysis.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

**Dataset:** `student-mat.csv`

The dataset contains information related to students' academic performance, study habits, and demographic characteristics, making it suitable for exploratory data analysis and visualization.

---

## 📈 Analysis Performed

* Data Loading
* Data Cleaning
* Missing Value Analysis
* Duplicate Detection
* Descriptive Statistics
* Average Final Grade (G3) Analysis
* Students Scoring Above 15
* Correlation Analysis (Study Time vs Final Grade)
* Gender-wise Performance Comparison
* Data Visualization

---

## 🔍 Key Findings

* Average Final Grade (G3): **9.8 / 20**
* Study time shows a **weak positive correlation** with academic performance (**r = 0.11**).
* Female students achieved a slightly higher average final grade (**10.17**) than male students (**9.48**).
* Academic performance is influenced by multiple factors beyond study time alone.
